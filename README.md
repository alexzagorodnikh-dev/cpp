# C++ Struct Tasks

## 📌 Опис
Цей репозиторій містить 20 задач на використання структур (struct) в C++.

Програма реалізована в одному файлі:
- struct_tasks.cpp

## ⚙️ Компіляція

g++ struct_tasks.cpp -o struct_tasks

## ▶️ Запуск

./struct_tasks

## 📚 Теми, які використані:
- struct
- функції
- switch-case
- робота з введенням/виведенням
- базова математика
